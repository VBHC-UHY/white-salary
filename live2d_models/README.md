# Live2D Models

This directory contains the default Live2D assets used by the desktop pet.

- `default/` is the model loaded by the desktop app by default.
- `backup_hiyori/` keeps a backup sample model for testing and replacement.
- `white_salary/` and `generated/` contain generated reference textures.

These model assets are provided for convenience so a fresh clone can show the
desktop character immediately. They are not part of the MIT-licensed source
code; use them according to their own source and license notes. Docker builds
still ignore this directory to keep server images small.
