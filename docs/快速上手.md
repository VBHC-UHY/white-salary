# 快速上手 —— 完全新手图文说明书

**这份说明书是写给"完全不懂编程"的你的。** 不需要知道 pip 是什么、不需要打过命令行。
全程只有四步：**下载 → 双击安装 → 粘贴一把密钥 → 启动**。每一步都会告诉你：

- 🖥️ **你会看到什么**（屏幕上的提示原文长什么样）
- 👆 **你该做什么**（照做就行）
- 🔧 **出问题看哪里**（对号入座）

> 想手动控制每一步的进阶用户请看 [INSTALL.md](INSTALL.md)；本文只走"最省心"的一条路。

---

## 第 0 步：把项目下载到你的电脑

两种方式任选其一：

### 方式 A：下载 ZIP（不会用 Git 就选这个）

1. 打开项目的 GitHub 页面（github.com/VBHC-UHY/white-salary）。
2. 点页面右上方的绿色 **`Code`** 按钮 → 在弹出的菜单里点 **`Download ZIP`**。
3. 下载完成后，右键 ZIP 文件 → **全部解压缩**，解压到一个**路径简单的文件夹**。

> ⚠️ **解压路径注意**：
> - 尽量**别放在中文名、层级极深的目录**里（比如 `D:\我的下载\新建文件夹(3)\临时\...`），个别工具遇到复杂路径容易出怪问题。推荐直接解压到 `D:\WhiteSalary` 这种简短路径。
> - 路径里**有空格没关系**（项目自己就叫 White Salary），只是以后自己敲命令时记得给路径加英文引号。

### 方式 B：git clone（会用 Git 的话）

```bash
git clone https://github.com/VBHC-UHY/white-salary.git "White Salary"
```

✅ **本步验收**：打开解压/克隆出来的文件夹，能看到 `安装.bat`、`Start.bat`、`conf.default.yaml` 这些文件，就对了。

## 第 1 步：双击 `安装.bat`

在项目文件夹里找到 **`安装.bat`**，双击它。会弹出一个黑色窗口（标题是"White Salary - 一键安装"），开头显示：

```
============================================================
  White Salary 一键安装
  只需要几分钟，装完就能和白聊天了
============================================================
```

然后它会自动走 5 步。你什么都不用点，看着它跑就行——下面告诉你每一步的屏幕提示长什么样、出问题怎么办。

> 💡 **先吃一颗定心丸**：这个脚本**重复运行是安全的**。装到一半失败了？直接再双击一次，已完成的步骤会自动跳过（术语叫"幂等"），不会装重、不会覆盖你已有的配置。

### [1/5] 检查 Python

- 🖥️ **正常时你会看到**：`[OK] 检测到 Python 3.11.x`（版本数字因人而异），然后自动进入下一步。
- 🖥️ **电脑上没有 Python 时你会看到**：

  ```
  [X] 没有找到 Python。请先到官网下载安装，选 3.11 或更新版本：
      https://www.python.org/downloads/
      ★ 安装第一步务必勾选 "Add Python to PATH"，不勾装完也找不到！
      装好 Python 后，再回来双击本脚本继续。
  ```

- 👆 **你该做什么**：
  1. 用浏览器打开 https://www.python.org/downloads/ ，点大大的黄色 **Download Python 3.x** 按钮下载安装包。
  2. 双击安装包，**在安装的第一个画面，把最下面的 `Add Python to PATH` 复选框勾上**——这是整个安装过程中**最容易踩的坑**，不勾的话装完了系统也"找不到" Python，一切白装。
  3. 点 Install Now，装完关掉安装器，**回来重新双击 `安装.bat`**。
- 🖥️ **Python 太旧时你会看到**：`[X] 你的 Python 版本太旧，需要 3.10 及以上（建议 3.11+）。` → 处理方式同上，去官网装个新版。

### [2/5] 检查 Node.js（可选，只影响桌宠小人）

- 🖥️ **正常时你会看到**：`[OK] 检测到 Node.js v20.x.x`。
- 🖥️ **没装 Node.js 时你会看到**：

  ```
  [!] 没有找到 Node.js —— 只影响桌宠小人界面，聊天后端不受影响。
      想要桌面小人的话，稍后到 https://nodejs.org 装个 LTS 版即可，
      装好后重新双击本脚本会自动补装前端依赖。现在先继续。
  ```

- 👆 **你该做什么**：**什么都不用做，安装会继续**。但既然玩桌宠肯定想要桌面小人——建议现在就开 https://nodejs.org ，点绿色的 **LTS** 版本下载安装（一路下一步即可，不用改任何选项），装完**重新双击 `安装.bat`**，它会自动把前端补上。

### [3/5] 安装 Python 依赖

- 🖥️ **你会看到**：`[3/5] 安装 Python 依赖（第一次要几分钟，请耐心等待）...`，然后满屏滚动下载进度——**这是正常的，去倒杯水**。装过的话则直接显示 `[OK] Python 依赖已经装好了，跳过。`
- 🖥️ **下载慢/失败时你会看到**（国内网络常见）：

  ```
  [X] 依赖安装失败。国内网络建议改用清华镜像重试——
      把下面这行整个复制到本窗口，回车执行：
      python -m pip install -e .[llm-openai] -i https://pypi.tuna.tsinghua.edu.cn/simple
      装完后重新双击本脚本即可。
  ```

- 👆 **你该做什么**：把那行 `python -m pip install ...` 命令**整行抄下来**，然后：在项目文件夹的窗口里，点一下**顶部地址栏**，输入 `cmd` 按回车——会弹出一个新的黑窗口——把命令**粘贴进去按回车**。用了清华镜像后下载会快很多。装完**重新双击 `安装.bat`**（放心，装好的部分会自动跳过）。

### [4/5] 安装桌宠界面依赖

- 🖥️ **你会看到**：npm 的下载滚动条（第一次要下载 Electron，稍等），最后 `[OK] 前端依赖安装完成。`；没装 Node.js 的话显示 `[跳过] 没装 Node.js，先跳过。装好 Node 后重新双击本脚本即可补上。`
- 🖥️ **失败时你会看到**：`[!] 前端依赖安装失败 —— 不影响后端聊天。` → 不阻塞安装，之后重新双击 `安装.bat` 重试即可。

### [5/5] 准备配置文件

- 🖥️ **你会看到**：`[OK] 已从模板生成 conf.yaml。` 和 `[OK] 已从示例生成 prompts\system_prompt.txt —— 白的人格设定。`（重复运行时则显示"已存在，保留你现有的配置"——**不会覆盖你改过的东西**）。

### 收尾：自动弹出配置向导

5 步全部走完，你会看到：

```
============================================================
  安装完成！马上会弹出配置向导 ——
  在窗口里粘贴你的 API Key 就大功告成了。
============================================================
```

紧接着屏幕上会**弹出一个小窗口**（标题"White Salary 配置向导"）——进入第 2 步。

> 🔧 **向导没弹出来 / 不小心关掉了？** 不用重装。在项目文件夹地址栏输入 `cmd` 回车，在黑窗口里执行 `python scripts\setup_wizard.py` 就能再次打开；或者干脆重新双击 `安装.bat`——已装好的步骤会秒过，直达向导。

## 第 2 步：配置向导（粘贴一把密钥）

向导一共三页，全程点两三下按钮。

### 第 1 页：欢迎页

- 🖥️ **你会看到**：大标题"**欢迎使用 White Salary**"，下面写着：

  > 白是一只住在你桌面上的 AI 伙伴。
  > 只需一把 API 密钥就能开始——下一步会告诉你去哪里免费领。

- 👆 **你该做什么**：点 **`开始配置 →`**。

### 第 2 页：粘贴你的 API Key（核心页）

- 🖥️ **你会看到**：标题"**粘贴你的 API Key**"，下面一行小字"推荐用「硅基流动」：注册就送额度，一把 key 聊天 / 看图 / 语音全通。"，一个**大输入框**，旁边一个 **`去注册 / 拿 Key`** 按钮；下方还有"高级选项（换一家供应商）▸"、"测试连通"按钮和"保存并继续 →"。

- 👆 **你该做什么（拿 key 三步走）**：
  1. 点 **`去注册 / 拿 Key`** 按钮——浏览器会自动打开**硅基流动的 API 密钥页**（https://cloud.siliconflow.cn/account/ak ）。没有账号的话先用**手机号注册**（免费，新账号通常送试用额度）。
  2. 登录后在"API 密钥"页面点 **新建 API 密钥**，然后点它**复制**——你会得到一串 `sk-` 开头的字符。这就是"key"，相当于你在这家服务的通行证，**别发给别人**。
  3. 回到向导窗口，把 key **粘贴进大输入框**（Ctrl+V）。

  然后**强烈建议**点一下 **`测试连通`**：
  - 🖥️ 先显示"测试中，请稍候（最多 30 秒）…"；
  - 🖥️ **成功**显示：`✅ 连通成功，耗时 x.x 秒` → 放心点 **`保存并继续 →`**；
  - 🖥️ **失败**显示：`❌ + 失败原因`（中文）。常见情况：key 复制少了几个字符（回去重新复制整串）、网络不通（换个网试试）；若显示 `❌ 依赖没装齐，暂时无法测试…请先运行 安装.bat 完成依赖安装`，说明第 1 步没走完，先回去重跑 `安装.bat`。

  最后点 **`保存并继续 →`**。key 没填就点保存的话，会弹窗提醒"还没填 Key"，不会写坏任何东西。

> ⚠️ **为什么只填一把 key 就能"全功能"跑起来？** 一句话原理：向导会把这把硅基流动的 key 同时写进"聊天"和"看图"两个通道，而"语音识别/语音合成"在没单独配密钥时会**自动复用**这把 key——所以聊天、发图给白看、语音对话，一把 key 全通。

> 💡 **不想用硅基流动？** 点"**高级选项（换一家供应商）▸**"展开下拉框，可换成 **DeepSeek 官方**（拿 key：https://platform.deepseek.com/api_keys ）、**Kimi（月之暗面）**（https://platform.moonshot.cn/console/api-keys ）或 **DMXAPI（Claude / GPT 中转）**（https://www.dmxapi.cn/token ）。下方小字会显示"将使用模型：xxx（地址已内置，无需手填）"——模型和地址都替你选好了。注意：换了别家，"一把 key 全通"的看图/语音福利就没有了（那是硅基流动专属），聊天照常。

### 第 3 页：完成页

- 🖥️ **你会看到**：大标题"🎉 **配置完成！**"，下面写着接入了哪家、用什么模型（选硅基流动时还有一句"（看图 / 语音也一并配好了）"），以及"双击项目文件夹里的 Start.bat 即可启动白。"外加三个按钮：
  - **`现在就启动白 🚀`** —— 替你直接把 `Start.bat` 跑起来（推荐直接点它）；
  - **`想解锁更多功能（生图 / QQ / B站）看这里`** —— 打开进阶文档 [EXTERNAL_SERVICES.md](EXTERNAL_SERVICES.md)；
  - **`完成，关闭窗口`**。

## 第 3 步：启动与验收

以后每次想找白，都是同一个动作：**双击项目文件夹里的 `Start.bat`**（刚才在向导点了"现在就启动白 🚀"的话，这次可以跳过）。

- 🖥️ **你会看到**：一个标题"White Salary - Launcher"的黑窗口，依次跑 `[0/4] Cleaning up old processes...` → `[1/4] Starting local TTS server (GPT-SoVITS)...` → `[2/4] Starting backend server...` → `[3/4] Checking frontend dependencies...` → `[4/4] Starting desktop app...`，期间会**另外弹出几个黑窗口**（后端、前端各一个）——**都不要关**，最小化即可。
- ⏳ **`[1/4]` 这步没装本地语音的人会卡约 45 秒**：它在尝试启动本地语音引擎 GPT-SoVITS（进阶玩家自装的东西），你没装的话会闪出一个很快报错的小黑窗并等待一会儿——**这是正常现象**，把那个报错小窗关掉、耐心等启动器继续走就行。白会自动改用云端语音，聊天完全不受影响。
- ✅ **验收**：桌面上出现白的 Live2D 形象（透明背景、悬浮置顶），对她说句话能得到回复。按 **`Ctrl+,`**（Ctrl 加逗号）可以打开**控制面板**——22 页设置都在里面。
- 🖥️ **还没配置就双击 Start.bat 的话**：会提示 `请先双击 安装.bat 完成初始化` —— 按提示回到第 1 步即可。

### 关于 Live2D 形象：GitHub 下载的项目"能聊天但没形象"是正常的

出于版权原因，**仓库不附带 Live2D 模型文件**（`live2d_models/` 目录不随源码分发）。所以从 GitHub 下载的项目启动后：**聊天窗口、语音、控制面板全部正常，只是屏幕上看不到小人形象**——不是装坏了。

想要形象，自备一个 Live2D 模型即可：

1. 去 **Live2D 官方免费示例模型页**下载：https://www.live2d.com/en/learn/sample/ （下载前请阅读并遵守页面上的许可条款，这些模型有各自的使用限制，**不可随意二次分发**）。
2. 在项目根目录新建 `live2d_models\default\` 文件夹，把模型解压进去（里面应该有一个 `xxx.model3.json` 文件）。
3. 用记事本打开 `frontend\js\app.js`，找到开头的 `LIVE2D_MODEL_PATH` 那一行，把路径改成你的模型文件，例如：
   `const LIVE2D_MODEL_PATH = '../live2d_models/default/你的模型名.model3.json';`
4. 重新双击 `Start.bat`（或按 `Ctrl+Shift+R` 重载前端）。

> 🔍 **不确定整体配好没？** 在项目文件夹的黑窗口里执行 `python scripts\first_run_check.py`，它会用中文逐项检查并告诉你"下一步做什么"。

## 常见问题（对号入座）

| 问题 | 答案 |
|------|------|
| **装到一半失败了，重新双击 `安装.bat` 安全吗？** | **安全，放心重跑。** 脚本是幂等的：已装好的依赖秒过、已存在的 `conf.yaml` 和人格文件都会保留不覆盖。失败几次就重跑几次，不会越装越坏。 |
| **key 填错了 / 想换一把，怎么改？** | 两条路任选：① 重跑向导——执行 `python scripts\setup_wizard.py`（或重新双击 `安装.bat`，装好的步骤会跳过直达向导），粘贴新 key 保存即可；② 启动后按 `Ctrl+,` 打开控制面板，在「LLM配置」页直接改。 |
| **语音没声音？** | 先确认你在向导里填的是**硅基流动**的 key（云端语音复用它；填的别家 key 则语音不通，需在 `conf.yaml` 的 `tts.fallback_api_key` 补一把硅基流动 key）。云端语音默认用**公共预置音色**（anna，温柔女声），开箱即用；想要白的**本地专属声音**需要自装 GPT-SoVITS，见 [LOCAL_ADVANCED.md](LOCAL_ADVANCED.md)。 |
| **QQ 机器人怎么开？** | 需要自己去官方仓库 [NapCat](https://github.com/NapNeko/NapCatQQ) 下载（第三方开源工具，**不随本仓库分发**，新手用一键版 NapCat.OneKey），登录 QQ、配好正向 WebSocket，再到 `conf.yaml` 打开 `qq.enabled`。完整步骤见 [INSTALL.md 的 6.2 节](INSTALL.md#62-qq-机器人napcat)。属于可选进阶，不开不影响桌宠。 |
| **配置向导窗口关早了 / 没弹出来？** | 执行 `python scripts\setup_wizard.py` 随时可以再打开；也可以重新双击 `安装.bat`。 |
| **想解锁生图 / 生视频 / B 站直播？** | 见 [EXTERNAL_SERVICES.md](EXTERNAL_SERVICES.md)（纯云端，填 key 就行）和 [LOCAL_ADVANCED.md](LOCAL_ADVANCED.md)（本地大模型进阶）。 |

装好之后想精细调参（8 个 LLM 通道逐个换模型、记忆、好感度……），按 `Ctrl+,` 打开控制面板慢慢逛，或看 [CONFIG.md](CONFIG.md)。祝你和白玩得开心。
